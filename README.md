# Chippy
A Chip-8 Emulator written in javascript

[Live DEMO](https://kennethlamb.me/Chip-8%20Emulator/index.html)

![Main Window](./images/Display.png)

# Features
* Display Controls
* CPU Controls
* Sound Controls
* Rom Selection

# Debug Features
