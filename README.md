# Chippy
A Chip-8 Emulator written in javascript

[Live DEMO](https://slurrps-mcgee.github.io/Chippy/)

![Main Window](./images/Display.png)

# Features
* Display Controls
* CPU Controls
* Sound Controls
* Rom Selection

# Debug Features
* Register Display
